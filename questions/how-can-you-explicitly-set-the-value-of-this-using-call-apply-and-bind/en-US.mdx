## TL;DR

You can explicitly set the value of `this` in JavaScript using the `call`, `apply`, and `bind` methods. `call` and `apply` immediately invoke the function with a specified `this` value, but `call` takes arguments individually while `apply` takes them as an array. `bind` returns a new function with a specified `this` value, which can be invoked later.

```javascript
function greet(greeting) {
    console.log(greeting + ', ' + this.name);
}

const person = { name: 'Alice' };

greet.call(person, 'Hello'); // Hello, Alice
greet.apply(person, ['Hi']); // Hi, Alice

const boundGreet = greet.bind(person);
boundGreet('Hey'); // Hey, Alice
```

## Using `call`, `apply`, and `bind` to set `this`

### `call` method

The `call` method immediately invokes the function with a specified `this` value and arguments provided individually.

```javascript
function greet(greeting) {
    console.log(greeting + ', ' + this.name);
}

const person = { name: 'Alice' };

greet.call(person, 'Hello'); // Output: Hello, Alice
```

### `apply` method

The `apply` method immediately invokes the function with a specified `this` value, but arguments are provided as an array.

```javascript
function greet(greeting) {
    console.log(greeting + ', ' + this.name);
}

const person = { name: 'Alice' };

greet.apply(person, ['Hi']); // Output: Hi, Alice
```

### `bind` method

The `bind` method returns a new function with a specified `this` value. This new function can be invoked later with arguments provided individually.

```javascript
function greet(greeting) {
    console.log(greeting + ', ' + this.name);
}

const person = { name: 'Alice' };

const boundGreet = greet.bind(person);
boundGreet('Hey'); // Output: Hey, Alice
```
